## Commodore 64 Ultimate (C64U)

### Controles y ajustes básicos

#### Multi Function Switch

Es un interruptor basculante que se puede pulsar o mantener hacia arriba o hacia abajo:

| Acción | Operación |
| --- | --- |
| Encender (estando apagado) | Pulsar arriba o abajo |
| Apagar (estando encendido) | Mantener abajo durante 4 segundos |
| Reset (fuera del menú) | Mantener arriba 1 segundo y soltar |
| Mostrar/ocultar menú C64U | Pulsar arriba |

#### Navegación del menú C64U

El menú C64U está accesible en todo momento. Al pulsar arriba el Multi Function Switch se pausa el programa en ejecución y se muestra el menú; al salir, el programa se reanuda.

| Tecla | Función |
| --- | --- |
| `W` / `S` | Mover la selección arriba / abajo |
| `D` | Abrir el submenú seleccionado |
| `A` | Volver al menú padre |
| `RETURN` | Seleccionar una opción o acción |
| `RUN/STOP` | Salir de un popup, volver al menú padre o salir del menú (pulsar repetidamente para salir rápido) |
| `F1` | Abrir el menú de herramientas (Tool Menu) |
| `F3` / `F5` | Página arriba / abajo |
| `F7` | Ayuda |
| `Y` / `N` | Confirmar / descartar el guardado de cambios al salir |

También se pueden usar las cursor keys del C64 para navegar.

#### Safe Mode

Si se cambia un ajuste incompatible con la pantalla y se pierde el acceso al menú, se puede arrancar en *safe mode*, que resetea temporalmente todos los ajustes a sus valores por defecto:

1. Mantener el Multi Function Switch abajo durante 4 segundos para apagar.
2. Mantener pulsada la tecla `RESTORE`.
3. Encender pulsando el Multi Function Switch arriba.

Para guardar los ajustes restaurados en flash, hacer cualquier cambio, salir del menú y guardar cuando se pregunte.

#### Modo de vídeo (PAL vs. NTSC)

Selección temporal del modo de vídeo al encender, manteniendo pulsadas estas teclas:

| Modo | Teclas |
| --- | --- |
| NTSC | `C=` + `N` |
| PAL | `C=` + `P` |

Para cambiar el modo de vídeo y guardarlo: menú C64U → *Video Setup* → *System Mode* (`RETURN`) → elegir *PAL* o *NTSC* → `RUN/STOP` para aplicar → `Y` para guardar.

#### Ajustes básicos del menú

| Ajuste | Ruta en el menú C64U |
| --- | --- |
| HDMI Scan Lines | *Video Setup* → *HDMI Scan Lines* → *Enabled* / *Disabled* |
| Altavoz interno | *Audio Setup* → *Speaker Mixer* → *Speaker Enable* → *Enabled* / *Disabled* |
| RAM Expansion Unit (REU) | *Memory & ROMs* → *RAM Expansion Unit* → *Enabled* / *Disabled* |
| Intercambio de joystick | *Joystick & Controllers* → *Joystick Input* → *Normal* / *Swapped* |

En general, hay que salir del menú para aplicar el ajuste y luego pulsar `Y` para guardarlo.

#### Solución de problemas de software

Algunas funciones (modo PAL/NTSC, REU, etc.) pueden hacer que ciertos programas se comporten mal. Cosas a probar:

* Cambiar el modo de vídeo de PAL a NTSC, o viceversa.
* Deshabilitar la RAM Expansion Unit.
* Si se han instalado las ROMs JiffyDOS, volver a las ROMs estándar del C64.
* Desconectar periféricos IEC sin usar (p. ej. unidades de disco externas).
* Deshabilitar la unidad de disco virtual B: *Disk Drive B Options* → *Drive* = *Disabled*.
* Deshabilitar Software IEC: Tool Menu (`F1`) → *Software IEC* → *Turn Off* (si está disponible).
* Deshabilitar la emulación de impresora: Tool Menu (`F1`) → *Printer* → *Turn Off* (si está disponible).

#### Escribir comandos

Al encender el C64U (sin cartucho) aparece el prompt `READY.` con el cursor parpadeando, listo para recibir comandos:

* Las letras aparecen en mayúsculas por defecto; no hace falta `SHIFT` para escribir una mayúscula.
* Para escribir las comillas dobles (`"`), mantener `SHIFT` y pulsar `2`.
* Para borrar un carácter, pulsar `INST DEL`.
* Pulsar `RETURN` (con el cursor en cualquier punto de la línea) para ejecutar el comando.

Si se llega a un estado confuso al escribir comandos, mantener `RUN/STOP` y pulsar `RESTORE`: la pantalla se limpia y aparece un nuevo prompt `READY.`.
